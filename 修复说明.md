# Merge Commit 渲染修复说明

## 问题
Merge commit 显示为 `[object Object]`，而不是自定义格式 `[分支标签] [hash] [作者名]: [提交信息] ([时间])`

## 解决方案
✅ 已通过 patch-package 修复 @gitgraph/core 库

## 使用方法

### 首次安装
```bash
npm install  # patch 会自动应用
npm run build
npm start
```

### 验证修复
启动后检查：
1. 普通提交显示格式正确
2. Merge 提交显示格式正确  
3. 可以看到分支之间的 merge 线条
