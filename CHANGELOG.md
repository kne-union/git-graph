# 更新日志

## [0.1.1] - 2026-03-30

### 新增
- 支持从 GitHub 获取所有提交历史（最多 10000 个，受 GitHub API 限制）
- 添加实时进度显示，显示当前获取状态和已获取的提交数量
- 自动分页获取所有分支和提交

### 改进
- 优化数据获取性能，使用分页机制
- 改进加载状态显示，提供更详细的进度信息
- 修复 author 对象显示为 `[object Object]` 的问题

### 技术细节
- `fetchGitDataFromGitHub` 函数现在支持进度回调
- 自动处理 GitHub API 分页
- 每页获取 100 条记录
- 最多获取 100 页（10000 条提交）

## [0.1.0] - 2026-03-27

### 新增
- 初始版本发布
- 实现三栏布局设计（分支列表 + 提交历史 + 提交详情）
- 使用 @gitgraph/react 绘制 Git Graph
- 支持分支选择和提交选择
- 支持提交筛选功能
- 支持国际化（中文/英文）
- 使用 antd Splitter 实现可拖拽面板
- 添加 GitDataFetcher 组件用于从 GitHub 获取数据
